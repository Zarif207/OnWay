A complete ride-sharing platform like Uber/Pathao with real-time tracking and role-based dashboards.

🔹 Key Features:

👤 Passenger ride booking system
🚗 Driver trip acceptance & earnings
🛡️ Support agent live chat & SOS handling
👨‍💼 Admin full system control
📍 Real-time location tracking (Google Maps API)
💳 Secure payments (SSLCommerz + Stripe)
🔐 Role-based authentication (NextAuth + Firebase)
